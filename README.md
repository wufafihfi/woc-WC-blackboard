# 沃C黑板
- it just a blackboard
- 有的时候希沃启动太慢或者出问题可以暂时用这个代替
- 功能不够丰富,所以凑合着用.
- GravitySimulaer这个名称的出现说明这个项目是走歪路的

## 此项目使用了一下开源库
- **SFML 3.0.1** - 采用 zlib/libpng 许可证  
  官网：https://www.sfml-dev.org/

- **Dear ImGui 1.91.9 (Docking分支)** - 采用 MIT 许可证  
  项目地址：https://github.com/ocornut/imgui

- **imgui-sfml 3.0** - 采用 MIT 许可证  
  项目地址：https://github.com/elisee/imgui-sfml

- **JsonCpp 1.9.6** - 采用 MIT 许可证  
  项目地址：https://github.com/open-source-parsers/jsoncpp

- JsonCpp 和 SFML 3.0.1使用VCPKG集成
- Dear ImGui 1.91.9 (Docking分支) 和 imgui-sfml 3.0在Dependence文件夹中
